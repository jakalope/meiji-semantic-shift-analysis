# Data Directory

This directory contains the text corpora and processed data for the Edo-Meiji polysemy analysis.

## Directory Structure

```
data/
├── README.md           # This file
├── edo/                # Edo period (1603-1868) texts
├── meiji/              # Meiji period (1868-1912) texts
├── processed/          # Preprocessed data (contexts, word frequencies)
└── embeddings/         # BERT embeddings (generated)
```

## Data Sources

### Aozora Bunko (青空文庫)
- Public domain Japanese literature repository
- URL: https://www.aozora.gr.jp/

### Recommended Texts

**Edo Period Authors:**
- Ihara Saikaku (井原西鶴, 1642-1693) - "The Life of an Amorous Man" (好色一代男)
- Jippensha Ikku (十返舎一九, 1765-1831) - "Hizakurige" (東海道中膝栗毛)
- Chikamatsu Monzaemon (近松門左衛門, 1653-1724) - Puppet plays

**Meiji Period Authors:**
- Natsume Sōseki (夏目漱石, 1867-1916) - "Kokoro" (こころ), "Botchan" (坊っちゃん)
- Mori Ōgai (森鷗外, 1862-1922) - "The Wild Geese" (雁)
- Higuchi Ichiyō (樋口一葉, 1872-1896) - "Growing Up" (たけくらべ)

## Preparing Your Data

1. **Download texts** from Aozora Bunko or other sources
2. **Save as UTF-8 encoded .txt files**
3. **Organize by era:**
   - Place Edo texts in `data/edo/`
   - Place Meiji texts in `data/meiji/`
4. **Remove metadata** (front matter, publication info) from text files
5. **Run preprocessing:**
   ```bash
   python src/data_preprocess.py --edo-dir data/edo --meiji-dir data/meiji --output data/processed
   ```

## Sample Data Format

Text files should be plain UTF-8 text:

```
吾輩は猫である。名前はまだ無い。
どこで生れたかとんと見当がつかぬ。
何でも薄暗いじめじめした所でニャーニャー泣いていた事だけは記憶している。
```

## Data Files Generated by Pipeline

After running the pipeline, you will have:

- `processed/edo_word_frequencies.csv` - Word frequency counts for Edo corpus
- `processed/meiji_word_frequencies.csv` - Word frequency counts for Meiji corpus
- `processed/edo_contexts.json` - Extracted sentences containing target words (Edo)
- `processed/meiji_contexts.json` - Extracted sentences containing target words (Meiji)
- `embeddings/edo_embeddings.pkl` - BERT embeddings for Edo words
- `embeddings/meiji_embeddings.pkl` - BERT embeddings for Meiji words

## Copyright and Usage

- Ensure all texts used are in the public domain or you have appropriate permissions
- Aozora Bunko texts are public domain in Japan
- Check copyright status for texts published after 1926
- For academic use, always cite original sources

## Notes

- Large data files (>100MB) are excluded from git via `.gitignore`
- For reproducibility, document which specific texts/versions you used
- Minimum recommended corpus size: 50,000 characters per era
- Ideal corpus size: 500,000+ characters per era for robust statistics
